Hexo+Github搭建个人博客

No tag
Publish Date:   2024-10-26

##学习课程
https://www.bilibili.com/video/BV1q741167PJ?spm_id_from=333.788.player.switch&vd_source=71317e740fa37f0c44917c26d27af26f&p=3

Hexo
官方网站https://hexo.io/zh-tw/
github：https://github.com/hexojs/hexo

环境安装
官方文档：https://hexo.io/zh-tw/docs/
下载GIthttps://git-scm.com/downloads/win
安裝 Node.js：https://nodejs.org/en/download/package-manager
安裝 Hexo（天坑！ npm 在国内下载非常非常慢， 需要设置 proxy 才能下载。）1.npm无法安装Hexo
一旦所有的必備軟體都安裝完畢後，即可透過 npm 安裝 Hexo：$ npm install -g hexo-cli

一旦 Hexo 完成後，請執行下列指令，Hexo 會在指定資料夾中建立所有您需要的檔案。
｜ https://hexo.io/zh-tw/docs/setup
    hexo init <自己选定的文件夹名>
    cd <自己选定的文件夹名>
    npm install
    注：这一步要注意路径，先cd跳转到你要建立文件夹的路径再hexo inti新建文件夹，如

运行hexo server，出现一个网址http://localhost:4000/，访问它，这就是当前运行的博客，基本框架生成好了

文件夹目录结构
打开vs code，运行ctrl+shift+E（侧边栏那个放大镜explore也可以）打开博客文件夹，在旁边显示目录
node modules是一些必要模块
source：用markdown写文章
themes：主题，可以做各种各样的配置
packages：使用到的包
最重要的是config

生成静态网站
生成静态网站hexo generate或hexo g：使用 Hexo 產生靜態檔案非常快速而且簡單。此时都生成了静态网站文件
｜ 注意先ctrl+c退出
这时候可以看到目录下多了一个文件夹public

一些hexo博客主题的安装与推荐
hexo官网上本身就有提供一些主题，选择喜欢的主题根据步骤安装
大部分安装步骤基本都是
先转到themes文件夹：cd themes
git clone https://github.com/…………（后面是主题库的地址）
｜ 例如安装闪烁之狐的主题：git clone https://github.com/blinkfox/hexo-theme-matery
｜ 闪烁之狐的个人页：http://blinkfox.com/
Configuration
Modify theme：Modify the value of theme in _config.yml of Hexo’s root folder: theme: hexo-theme-matery.然后保存

打造hexo主题
    在 themes 資料夾內，新增一個資料夾，並修改 _config.yml 內的 theme 設定，即可切換主題。 
    To start using your theme, modify the theme setting in your site’s _config.yml. 
    一個主題可能會有以下的結構：
        .
        ├── _config.yml
        ├── languages
        ├── layout
        ├── scripts
        └── source


启动一下看看效果
    hexo server
    打开给出的网址

自定义你的博客结构
对目前themes各部分内容的详细说明：vs code左边目录里看到刚才下载好的主题下有各个组件，like this（如图），需要修改哪些部分就在对应的文件下修改代码，这里你可能需要一些基本的网络开发知识技术（如CSS），基于模板文件进行修改（编写、样式设定，，做出自己喜欢的网页
